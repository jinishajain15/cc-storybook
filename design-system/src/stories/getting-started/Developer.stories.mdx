        # Getting Started Guide for Developers
    </div>

    The TruShield Design System contains a collection of Angular components that can be used to build accessible and performant Angular applications consistent with TruShield brand guidelines.
    This getting started guide provides the basic instructions to install the TruShield Design System components as a dependency in your application and begin using those components.
    The documentation for each individual component provides more detailed information including:

    + An overview with contextual usage guidance and best practices
    + Examples of common variations of the component such as sizes and states
    + Guidance for using the components with accessiblity in mind
    + A detailed description of available Angular props along with an interactive playground environment
